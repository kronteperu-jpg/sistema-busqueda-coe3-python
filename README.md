# 🚔 Sistema Policial y Serenazgo

Sistema en Python para búsqueda, registro y edición de efectivos policiales y serenazgo.

## 🔎 Funciones
- Buscar personal por nombre o DNI
- Registrar nuevos efectivos
- Editar información (CIP, rango, etc.)
- Listar base de datos completa

## 💻 Lenguaje
- Python 3
